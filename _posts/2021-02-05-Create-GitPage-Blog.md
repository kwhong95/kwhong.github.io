> GitHub Blog 서비스로 활용중인 `github.io` 블로그를 시작해보자.

GitHub Blog 서비스의 이름은 Pages이고 무료로 배포까지 해주니 너무나 감사한 서비스라고 할수 있다.

이미 상당히 많은 분들이 사용하고 있고 나는 공부하는 내용을 정리해놓고 개발 일지의 개념으로 활용하고 싶어 시작하게 되었다.
